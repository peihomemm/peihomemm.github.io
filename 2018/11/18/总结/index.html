使用hexo+github搭建博客

电脑系统：window 10 家庭版，64位

##1.安装node.js（最新版）和node.js环境 ##

成功界面

2.安装Git和Git环境安装成功的象征就是在电脑上任何位置鼠标右键能够出现如下两个选择

不懂看下面链接或者自行百度
https://jingyan.baidu.com/article/020278117cbe921bcc9ce51c.html

3注册Git账号

到github官网注册账号（过程有点麻烦，但还是可以的）

配置SSH Key

注册完之后需要添加 SSH Key。
SSH Key是一个认证，让github识别绑定这台机器，允许这台机器无需密码提交，修改项目。执行如下命令：

cd~/.ssh

~这个符号，表示在用户目录下
执行代码如果提示：No such file or directory 说明你是第一次使用git。

生产新的SSH Key配置

在Git Bash(在任意文件夹中鼠标右击选择Git Bash Here打开)执行代码

ssh-keygen -t rsa -C “自己的邮箱号”

成功后会生成个文件id_rsa（私钥）。然后找到个文件，默认都在C盘 > 用户 > xxx > .ssh 里面 （xxx是计算机用户名字）
然后用记事本把id_rsa这个文件打开，全选复制出来；
然后打开在github上添加SSH Key，登录github账号，点击右上角用户头像，选择setting（设置）> SSH and GPG keys > NewSSH key
然后把刚才复制的内容粘贴到key这里就可以了，title填不填都行。

Github新建项目

项目必须要遵守格式：账户名.github.io，不然接下来会有很多麻烦。并且需要勾选Initialize this repository with a README

在建好的项目右侧有个settings按钮，点击它，向下拉到GitHub Pages，你会看到那边有个网址，你将会惊奇的发现该项目已经被部署到网络上，能够通过外网来访问它。

##安装hexo
我是在D盘建了一个blog文件夹。然后通过Git Bash进入到该文件夹里面 输入npm install hexo -g，开始安装Hexo。

输入hexo -v，检查hexo是否安装成功

输入hexo init，初始化该文件夹（必须文件夹是空的）

看到后面的“Start blogging with Hexo！就是成功了”

输入npm install，安装所需要的组件（会出现一些代码，没有的应该是出问题了）我就是网络代理出现了问题。

有问题可以试试npm config get registry

输入hexo g，首次体验Hexo

输入hexo s，开启服务器，访问该网址，正式体验Hexo

问题：假如页面一直无法跳转，那么可能端口被占用了。此时我们ctrl+c停止服务器，接着输入“hexo server -p 端口号”来改变端口号


出现如上图就成功了

将Hexo与Github page联系起来，设置Git的user name和email（如果是第一次的话）

git confing –global user.name “用户名”

git config –global user.email “邮箱号”

输入cd ~/.ssh，检查是否由.ssh的文件夹

输入ls，列出该文件下的内容。
出现id-rsa …….. known-hosts就成功了

输入ssh-keygen -t rsa -C “91797406500@qq.com”，连续三个回车，生成密钥，最后得到了个文件：id_rsa和（默认存储路径是：C:\Users\Administrator.ssh）。

y继续

输入eval “$(ssh-agent -s)”，添加密钥到ssh-agent

再输入ssh-add ~/.ssh/id_rsa，添加生成的SSH key到ssh-agent

登录Github，点击头像下的settings，添加ssh

新建一个new ssh key，将id_rsa文件里的内容复制上去

输入ssh -T git@github.com，测试添加ssh是否成功。如果看到Hi后面是你的用户名，就说明成功了

配置Deployment，在其文件夹中，找到_config.yml文件，修改repo值（在末尾）

repo值是你在github项目里的ssh（右下角）

新建一篇博客，在cmd执行命令：hexo new post “博客名”

这时候在文件夹_posts目录下将会看到已经创建的文件

在生成以及部署文章之前，需要安装一个扩展：npm install hexo-deployer-git –save

使用编辑器编好文章，那么就可以使用命令：hexo d -g，生成以及部署了

署成功后访问你的地址：http://用户名.github.io。那么将看到生成的文章
hexo+github搭建博客完结
